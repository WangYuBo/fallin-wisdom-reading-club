## 常见问题
1. 如何报名成为分享人、轮值主席？
填写[分享人招募令](http://u1738750.viewer.maka.im/pcviewer/9JNTBN9Q)，负责人会第一时间联系你～

2. 如何成为核心成员？
满足其一即可：
- 累计参加过六次及以上读书会活动;
- 做过某期某主题分享人；
- 做过某期轮值主席；
